Reviewer Summary

  • c1ctl supersedes axiomctl as the durable Rust control CLI.
  • Rust now owns the first non-Rofi hey foundation slice: path, which, help, dynamic resolver, and exec.
  • Mutating command families remain delegated to Janet hey.
  • @rofi is an exact delegation boundary; traversal bypasses are rejected and tested.

In scope

  • packages/c1ctl package rename and implementation.

  • Rust foundation dispatcher/path/help/which/exec behavior.
  • Legion docs and wiki current truth.

Out of scope

  • Deleting Janet hey.
  • Porting Rofi implementation.

  • Live system mode switching during validation.
