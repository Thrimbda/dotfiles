What Changed / What Was Decided

在 hosts/aliyun-acorn/default.nix 中新增 host-level substituter：

nix.settings.substituters = lib.mkBefore [
  "https://mirrors.tuna.tsinghua.edu.cn/nix-channels/store"
];

选择 lib.mkBefore 是为了调整优先级而不覆盖既有 fallback。没有采用 lib.mkForce，因为国内 dynamic cache 缺 nar 时需要回退到 Cachix 或官方 cache。
