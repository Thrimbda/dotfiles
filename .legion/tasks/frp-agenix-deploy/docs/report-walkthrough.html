Reviewer Summary

  • 新增 modules.services.frp，支持 Linux 上的 frps 和 frpc systemd service。
  • aliyun-acorn 运行 frps 并开放 TCP 7000 与 2225。
  • axiom 运行 frpc，连接 8.159.128.125:7000，将本机 SSH 暴露为远端 2225。

What Changed / What Was Decided

frp module 使用 TOML 模板和运行时渲染，而不是把 token 写进 Nix 配置。review 后移除了无效的 age-secrets-frp-token.service 依赖，当前服务只排序在 network-online.target 之后。

保留 autossh reverse SSH 是显式 non-goal，frp 是新增部署路径，不是替代迁移。

Verification / Review Status

  • Secret consistency: PASS，两个 age 文件解密为同一个 96 位 hex token，未打印明文。
  • Nix eval: PASS，axiom 和 aliyun-acorn 均生成 toplevel drv path。
  • Dry-run build: PASS，两台 host 均完成。
  • frp verify: PASS，frpc 和 frps 模板均 syntax ok。
  • Review-change: PASS，security lens 已应用，无 blocking finding。

Risks and Limits

  • 公网暴露 7000 和 2225 是有意范围，安全边界依赖 token 与 SSH key-only auth。
  • 本地无法读取 axiom host 私钥，因此未直接用该私钥解密；recipient 已写入。
  • 真实网络连通性和 systemd runtime health 需要部署后在目标机确认。
